Fresh Market Database Final Project
Project Overview

This project was created as a final project for my Database Management course at Pierce College. The purpose of this project is to design and implement a SQL Server database for a local fruit and vegetable shop that sells products both in-store and online.

The database allows customers to purchase fruits and vegetables, tracks inventory, generates receipts, and ensures that product quantities cannot become negative.

Objectives
Create a relational database with at least five tables.
Maintain data integrity using constraints, triggers, and transactions.
Generate receipt totals using SQL views and aggregate functions.
Implement database security through user permissions.
Demonstrate SQL Server database design and management skills.
